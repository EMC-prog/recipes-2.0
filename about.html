Sobre EMC Recetas

EMC recetas es una simple demostración sobre las aplicaciones web progresivas, una alternativa a las aplicaciones normales.
El código fuente se puede encontrar en GitHub para su reutilización. EMC Recetas está bajo una licencia MIT. ¡Disfruten!

Registro de cambios
